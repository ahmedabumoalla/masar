# منصة مسار

منصة زراعية ذكية متعددة الصفحات مبنية باستخدام Next.js و React و TypeScript

## التشغيل المحلي

```bash
npm install
npm run dev
```

## الرفع على Vercel

ارفع المشروع على GitHub ثم اربط المستودع في Vercel

## مسار الشعار

```text
public/logo-masar.png
```

## الصفحات

- الصفحة الرئيسية
- المتجر
- تفاصيل المنتج
- السلة
- الدفع
- الاستشارات الزراعية
- طلب تركيب الأجهزة والخدمات الميدانية
- لوحة مؤشرات مختصرة

## صور المنتجات
تم استبدال صور المنتجات بروابط من Wikimedia Commons وهي منصة ملفات حرة ومفتوحة الاستخدام حسب ترخيص كل ملف داخل صفحة المصدر
تم اختيار الصور لتطابق نوع المنتج قدر الإمكان مثل نقاط الري وخطوط التنقيط والري المحوري والمرشات والمضخات ومحطات الضخ والأسمدة ومعدات حماية المحاصيل

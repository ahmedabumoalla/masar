# Masar Platform

منصة مسار الزراعية الذكية

## التشغيل المحلي

```bash
npm install
npm run dev
```

## الرفع على Vercel

المشروع جاهز للرفع مباشرة على Vercel بعد رفعه على GitHub

## المنتجات الحالية

- وحدة تحكم الري الذكي Masar Control X2
- مستشعر رطوبة التربة Soil Sense Pro
- حزمة ري بالتنقيط للمزارع الصغيرة Masar Drip Kit
- عداد تدفق ذكي Flow Track
- صمام كهربائي للتحكم بالقطاعات Sector Valve
- مغذيات زراعية متوازنة للنمو Balanced Grow
